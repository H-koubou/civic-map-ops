# 競合・市場調査メモ

## 既存類似サービス

### 1. 5374.jp (ゴミナビ)
- **URL**: https://5374.jp/
- **概要**: 全国のゴミ収集日を表示するOSSプロジェクト
- **強み**: ゴミ特化、軽量、複数自治体対応
- **弱み**: 地図機能弱い、下水道・施設情報なし
- **学べる点**: 自治体ごとのフォーク戦略、JSONデータ構造

### 2. 各自治体の公開GIS（ArcGIS Hub）
- 伊丹市「たみまっぷ」、芦屋市、西宮市等が同様のArcGIS Hubを運用
- **共通課題**: 業務UIで市民に使いにくい、モバイル最適化弱い

### 3. ハザードマップポータルサイト（国交省）
- **URL**: https://disaportal.gsi.go.jp/
- **学べる点**: レイヤ切替UI、国土地理院との統合方法

## 想定される伊丹市の課題（仮説）
1. **環境クリーンセンターへの問い合わせ多発**: 「これ何ゴミ？」「次の収集日いつ？」
2. **転入者の増加**: 大阪空港隣接で利便性高く転入多い → ゴミルール周知が常時必要
3. **既存たみまっぷの市民認知度低い**: ArcGISのUIで直帰率高い可能性
4. **多言語対応の遅れ**: 外国人住民向けの情報提供

## 提案時のアピールポイント
- 「市民問い合わせを減らす」具体的KPI提示
- 「既存たみまっぷを置き換えるのではなく市民向けの窓口として補完」
- 「他自治体への横展開で兵庫県全域モデルへ」
- 「OSS・公開データ中心でランニングコストが極小」

## 質問項目（市役所訪問時）
- 環境クリーンセンターへの問い合わせ年間件数
- ゴミ関連の苦情・相談の傾向
- たみまっぷの月間アクセス数（公開していれば）
- 市道側溝データの公開可否
- ゴミ収集ガイドブックの構造化データ提供可否
- デジタル戦略課の予算規模・既存ベンダー
